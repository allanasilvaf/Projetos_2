# Gambot UFPA

Assistente virtual inteligente para consulta de regulamentos e grades curriculares da Universidade Federal do Pará.

##Funcionalidades

- **Busca tradicional**: Procura por palavras-chave em PDFs
- **IA integrada**: Respostas inteligentes usando GPT-4o-mini
- **Sinônimos automáticos**: Entende variações de termos
- **Interface web**: Fácil de usar com Streamlit
- **Citações**: Mostra fonte dos documentos
